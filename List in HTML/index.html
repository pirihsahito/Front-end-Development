Definition List

HTML
HTML stands for HyperText Markup Language. It is the standard markup language used for creating web pages and web applications. Essentially, HTML provides the structure and content of a webpage
CSS
CSS, or Cascading Style Sheets, is a style sheet language used to describe the presentation of a document written in a markup language, most notably HTML. CSS dictates how that content is visually presented to the user.
JavaScript
JavaScript is a versatile, high-level programming language that is a core technology of the World Wide Web, alongside HTML and CSS. Originally designed to create interactive effects within web browsers, its use has since expanded to a wide range of applications, including mobile apps, games, and server-side development.
