Semantic Tags in HTML

Semantic HTML tags are elements that clearly describe the meaning and purpose of the content they contain to both the browser and the developer. Unlike non-semantic elements like

and , which offer no context, semantic tags provide a meaningful structure to the webpage, which improves accessibility, SEO, and code readability.

Common Semantic HTML5 Tags

header: Represents introductory content or a set of navigational links for a page or section.
nav: Defines a block of major navigation links.
main: Signifies the dominant, unique content of the document.
section: Groups thematically related content into a standalone section.
article: Used for self-contained content that could be distributed independently, like a blog post or news story.
aside: Marks content that is tangentially related to the main content, such as sidebars.
figure & figcaption: Group an image, diagram, or code block with its descriptive caption.
time: Used to define a specific date or time.
footer: Contains information like copyright, contact links, or site maps at the end of a page or section.
